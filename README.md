# Jornada de Programacao - Anderson Ferreira Santos

Repositorio dedicado ao meu aprendizado de programacao, seguindo um plano de estudos de 90 dias para me tornar um desenvolvedor no-code/low-code profissional.

## Sobre o Projeto

- **Inicio:** Janeiro 2025
- **Duracao:** 90 dias
- **Dedicacao:** 1-2 horas diarias
- **Objetivo:** Criar projetos funcionais e visualmente profissionais
